# Example: Earl Grey Bergamot Ice Cream (CG-005)

A real palace hotel recipe converted to a cooker-robot execution profile.

- `prompt.md` — the exact request sent to the agent
- `result-redacted.md` — the emitted cooker-class JSON profile with commentary
- `CG-005-cooker.json` — the actual emitted file

Why this recipe is a good test case: it mixes delicate constraints (bergamot
infusion capped at 60°C — the aroma degrades above), a pasteurization gate
(85°C / 30s), a long hold (18h maturation at 4°C), and a churning step to
-6°C. A converter that survives this recipe handles most of pastry.

The recipe is published with the owner's consent (the builder is the chef who
developed it).
