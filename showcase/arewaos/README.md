# ArewaOS

ArewaOS is a seven-layer autonomous Web3 intelligence agent built on Base by Zahra Usman ([@0xarewah](https://x.com/0xarewah)) from Kano, Nigeria.

The package isolates the ArewaOS build story, agent manifest, offerings, and market-action receipts under `showcase/arewaos/` so the repository root stays clean.

## What ArewaOS Does

- **Marketing Oracle:** campaigns, community audits, KOL mapping, grant applications, and FUD responses.
- **Storytelling Engine:** origin stories, wins-and-losses threads, and brand narratives with a human voice.
- **Crypto Confessions:** anonymous first-person confession content with specific amounts and real outcomes.
- **On-Chain Trading Analyst:** pre-pump fingerprints, structured technical/fundamental/sentiment review, stop loss, and NFA framing.
- **Ecosystem Scout:** grants, bounties, ambassador programs, testnets, and build opportunities ranked by effort-to-reward.
- **AI Training Specialist:** judgment-skill guidance for response ranking, rating, hallucination review, classification, and summarization work.
- **Prediction Market Analyst:** data-only probability work for event markets and edge calculation versus market price.
